## fedora-installer
Fedora linux installer script from KDE live CD.  
Internet is required. Run from a shell inside the GUI.  
You can customise the username, password, partitions (EFI/root/swap).  
Locale is set to UK/British English, timezone to London.  
Script copies mounted live root with rsync.  
Some configuration files, software and colour themes for KDE are added:
- RPM Fusion free and non-free repos
- more audio/video codecs
- htop mc iotop vainfo vim intel-media-driver kitty
